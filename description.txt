This mod provides relic coloring for the following mods:
Calamity's Vanities
Hunt of the Old God

Dormant Dawn
Calamity: Wrath of the Gods

It can take effect when used together with the above mods.
You can switch whether to color the mod relics in the mod settings.
This mod will continue to be updated to provide holy object coloring for more mods.
